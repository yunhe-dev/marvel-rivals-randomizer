### 新功能

- **更新日志页面**：新增由 content collections 驱动的更新日志，支持 Markdown release note
- **发布卡片**：为每个版本提供卡片布局，展示版本号、日期和正文内容
- **导航入口**：在导航栏 Pages 和页脚 Resources 中加入更新日志链接

### 技术改进

- 新增 `changelog` 内容集合，字段包括 title、description、date、version、published 和 content
- `getReleases()` 会按日期倒序排序，并过滤未发布内容
- 提供可复用的 `ReleaseCard` 组件来渲染 Markdown 内容

### 问题修复

- 使用 `release.slug` 作为更新日志列表的稳定 key
- 修正 TanStack Router 的路由 id：`/(pages)/changelog`
