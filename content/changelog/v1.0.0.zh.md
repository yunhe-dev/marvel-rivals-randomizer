### 核心功能

我们很高兴发布第一个正式版本，包含以下核心功能：

- **用户认证**：安全的登录与注册流程
- **仪表盘**：用于管理资源的直观控制台
- **项目管理**：创建、编辑和组织你的项目
- **API 集成**：通过 API 连接第三方服务

### 技术改进

- 基于 TanStack Start 和 React 构建，提供良好性能
- 集成 Drizzle ORM，实现类型安全的数据库操作
- 增加完整的错误处理和日志记录
- 针对移动端和桌面端体验进行优化

### 问题修复

- 修复用户注册流程中的问题
- 解决认证 token 过期处理问题
- 改进表单校验和错误提示
